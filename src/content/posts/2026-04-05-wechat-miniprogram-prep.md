## 一、账号注册

1. **准备一个邮箱**（每个邮箱只能注册一个小程序，注册前确认该邮箱未被使用过）
2. 前往 [微信公众平台](https://mp.weixin.qq.com) → 立即注册 → 选择「小程序」
3. 用邮箱注册账号，验证后设置密码
4. 填写主体信息：
   - 国家：大陆
   - 主体类型：个人（需要身份证、姓名）

注册成功后会显示提交成功提示：

![注册信息提交成功](/img/in-post/post-wechat-miniprogram-prep/07_注册信息提交成功_5min.png)

## 二、小程序基本信息填写

注册成功后，进入小程序管理后台填写：

- **名称**：小程序名称（备案时建议名称笼统一些，避免过于具体）
- **简称**：较短的名称
- **头像**：准备好小程序头像图片
- **介绍**：小程序功能描述
- **类目**：根据实际功能选择（如「工具」→「记录类」），类目选择直接影响审核通过率

## 三、合规性准备（个人开发者必看）

个人开发者资质有限，上线前需提前规避以下风险：

| 注意事项 | 说明 |
|----------|------|
| 类目选择 | 选「工具/记录」类，避免选医疗、金融等需资质认证的类目 |
| 避免专业词汇 | 不出现「诊断、治疗、医疗建议」等词汇，否则审核大概率不通过 |
| 明确工具属性 | 描述时强调是「记录工具」，不是医疗产品 |
| 个人信息保护 | 遵守个人信息保护法，用户数据不得滥用 |

## 四、小程序备案

> ⚠️ 这是上线最关键的前提条件，个人开发者容易忽略

- 和域名备案一样，小程序、App、网站上线前都需要备案
- **没有备案就无法提交审核，也无法发布上线**
- 备案流程：在微信公众平台 → 设置 → 基本设置 → 小程序备案

建议在开始开发之前就启动备案流程，因为备案审核本身需要时间。

## 五、开发工具准备

| 工具 | 用途 | 获取方式 |
|------|------|----------|
| 微信开发者工具 | 小程序本地开发、调试、上传 | 微信官方下载 |
| APPID | 开发时绑定小程序身份 | 微信公众平台 → 开发管理 → 开发设置 |
| 云环境 ID | 使用微信云开发时需要 | 微信公众平台 → 云服务 → 云开发 |

在微信开发者工具中创建项目时需要填入 APPID，并选择后端服务类型：

![微信开发者工具创建项目](/img/in-post/post-wechat-miniprogram-prep/09_微信开发者工具_APPID_9.55min.png)

## 六、云服务开通（如使用微信云开发）

1. 进入微信公众平台 → 开发与服务 → 云服务 → 云开发
2. 点击开通，用微信扫码授权
3. 创建云环境，**记录云环境 ID**（开发和部署时都会用到）

微信公众平台云开发开通页面：

![云服务开通](/img/in-post/post-wechat-miniprogram-prep/06_云服务开通_8min.png)

## 七、开发前资料准备

在正式写代码前，建议提前准备好：

- **需求文档**：明确功能模块、页面结构、数据字段（Claude Code 会直接读取这份文档生成代码）
- **设计规范**：如有 UI 要求，提前确定风格（如果没有，可以让 AI 自由发挥后再迭代）
- **图标/图片资源**：优先使用开源图标（如 iconfont），避免引用不存在的本地资源导致报错

准备好资料后，在 Claude Code 中使用 `/init` 命令初始化项目，生成 CLAUDE.md 和技术文档：

![Claude Code项目初始化](/img/in-post/post-wechat-miniprogram-prep/05_云开发配置与项目初始化_9min.png)

## 总结：上线前检查清单

```
□ 邮箱注册账号完成
□ 主体信息（身份证）登记完成
□ 小程序名称/头像/介绍填写完成
□ 类目选择合规
□ 小程序备案已提交（或进行中）
□ APPID 已记录
□ 云环境已开通，云环境 ID 已记录
□ 微信开发者工具已安装
□ 需求文档已准备
```
