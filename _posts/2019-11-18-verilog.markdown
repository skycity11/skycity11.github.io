# 1 verilog常用模块
## 1.1 上升沿及下降沿检测
上升沿
```verilog
wire	variable_rise;
reg		variable_r1;
reg		variable_r2;

assign variable_rise = ~variable_r2 & variable_r1;	//检测上升沿

always @(posedge clk_50m or negedge rst_n) begin
	if(!rst_n) begin
		variable_r1 <= 1'b0;
		variable_r2 <= 1'b0;
	end
	else begin
		variable_r1 <= variable;
		variable_r2 <= variable_r1;
	end
end
```
下降沿
```verilog
wire	variable_fall;
reg		variable_r1;
reg		variable_r2;

assign variable_fall = variable_r2 & ~variable_r1;	//检测下降沿

always @(posedge clk_50m or negedge rst_n) begin
	if(!rst_n) begin
		variable_r1 <= 1'b0;
		variable_r2 <= 1'b0;
	end
	else begin
		variable_r1 <= variable;
		variable_r2 <= variable_r1;
	end
end
```
